# EduPath - AI Powered Adaptive Learning Companion
### "Learning that Learns You Back"
Built on Microsoft Azure AI Foundry

---

## Setup Instructions

### Prerequisites
- Node.js v20+
- Python 3.x
- Azure AI Foundry project with a deployed chat model (gpt-4o-mini recommended)

### 1. Install dependencies

```bash
cd server && npm install
cd ../client && npm install
```

### 2. Configure environment variables
Create a `.env` file inside the `server/` folder:

AZURE_AI_ENDPOINT=endpoint

AZURE_AI_KEY=your_api_key_here

AZURE_AI_MODEL_DEPLOYMENT=your_deployment_name

### 3. Seed the database

```bash
cd server
npx tsx src/db/seed.ts
```

### 4. Start the backend

```bash
cd server
npx tsx src/index.ts
```
Server runs on http://localhost:3001

### 5. Start the frontend

```bash
cd client
npm run dev
```
Frontend runs on http://localhost:5173

|---|---|---|
| GET | /api/students | List all demo students |

## Key AI Features

### 1. Personalized Study Plan
Azure AI Foundry analyzes the student's knowledge gaps and mastery scores,
then generates a structured 5-day study plan tailored to their learning style,
with specific resources and reasoning for each day.

### 2. Doubt Resolution Chat
Context-aware AI tutor that knows the student's current topic, grade level,
and learning style. Uses the Socratic method where appropriate — guiding
students to think rather than just giving answers.

---

## Fallback Behavior
Both AI features include hardcoded fallback responses. If Azure AI Foundry
is unavailable during the demo, the app gracefully shows a pre-written plan
or response — the demo never breaks.
